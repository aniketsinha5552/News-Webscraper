## API Endpoints
- GET `/news` 
Get all the news from all the sources

- GET `/news/:source ` :
Get all the news from a particular source (toi/ thehindu/ cnn/ aljazeera)

- POST `/news/search/:query` 
Search news
Body: {search: "query"}
